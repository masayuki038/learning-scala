# Step 1. Learn to use the Scala interpreter
- Scalaを始める最も簡単な方法は、Scalaの記法やプログラムを書くためのシェルであるScalaインタプリタを開始することだ
- インタプリタにコードを書くと、評価されて結果が表示される
- このシェルは単にscalaと呼ばれる
- '1+2'のようにタイプしてエンターキーを押すと、

```
scala> 1 + 2
```

- インタープリタは以下のように出力する

```
res0: Int = 3
```
